JPA Auditing

This is a simple project to demostrate one way to implement auditing in our entities.

Key concepts:

The most common fields that we use when we are talking about auditing are:

- createdBy
- updateBy
- createdAt
- updatedAt

Technologies:

- Spring Boot
- Spring Data JPA
- Spring MVC
- Lombok
- Java 17 (coretto)
- H2 Database
- Gradle
